# Requirement
- PHP 7.2
- Mysql 8
- Laravel 6.0
- Redis
- npm or yarn
- bower

# Setup

## 2. Composer

```
composer install
```

Tạo file env và thiết lập, sau đó chạy

```
php artisan key:generate

```

Migrate

```
php artisan migrate
```

Seed

```
php artisan db:seed
```

Bower

```
bower update
```

Laravel mix

```
npm install
npm run dev
```
hoặc
```
yarn
yarn run dev
```
